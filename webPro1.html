About the Sorting Program

This web based program is created entirely with Html and JavaScript. It was designed to teach students new to algorithms about some of the most common sorting algorithms in computer science. The program is able to do this by providing a visual reference of the data being worked on as well as a red box signifying where actions are currently taking place. Along with a short description about each algorithm this program is designed to show how those algorithms work behind the scene to sort a given set of data.

The program visualizes data elements with randomly sized bars. These bars are then sorted by height from shortest to tallest. Each sorting algorithm has been broken down to itâ€™s most basic step so that the data can be progressed through the algorithm one iteration at a time. Some of the algorithms, like merge sort, were not designed to be an in place algorithm as shown in the picture representation. So, some slight changes needed to be made to the classic algorithm in order to make it work with the parameters the program has.
